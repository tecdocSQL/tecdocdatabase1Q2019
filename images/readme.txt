Это небольшая часть изображений товаров. Всего в базе изображений 5 024 455 штук, это около 95 гигабайт в архиве. Имена папок - это ID поставщиков товаров(Supplier), дочерние папки соответствуют первым двум символам названия файла изображения, указанному в таблице article_mediainformation, поле PictureName.
Для обеспечения возможности использования файлов изображений в файловой системе windows, к названиям файлов применены правила замены символов: " " > "==", "\"" > "==", "/" > "==", "\\" > "==", "*" > "==", ":" > "==", "|" > "==", ">" > "==", "<", "==", "?" > "=="
PictureName это по сути ID изображения, файл может называться AO-123.BMP но это всё равно изображение в формате JPG. Файлы сохранены с качеством JPG 80.
