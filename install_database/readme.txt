В данном файле описан процесс загрузки базы на СУБД MySQL.
На мой взгляд загружать данные из csv файлов проще всего используя для этого стандартный консольный клиент mysql.
Запускать mysql клиент следует с параметром -local-infile=1 например: mysql -h127.0.0.1 -uusername -ppassword -local-infile=1
На сервере стоит проверить переменную @@local_infile, должна быть 1 (select @@local_infile), если 0 установите (SET GLOBAL local_infile = 'ON')
1) Создайте новую базу данных с нужным Вам названием
2) Создайте структуру базы данных с помощью файла db.sql
3) В файле load_data.txt замените {database-name} на название базы которую Вы создали в первом пункте, {data-path} замените на путь к csv файлам например e:\\data\\ (двойные обратные слеши если используете консольный mysql клиент)
4) Вставьте содержимое отредактированного файла load_data.txt в mysql клиент и дождитесь окончания импорта

В файле query_examples.sql смотрите запросы демонстрирующие структуру базы.
